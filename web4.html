Java Script

Java Script란 자바스크립트는 객체 기반의 스크립트 언어로 웹의 동작을 구현한다.
이 언어는 웹 브라우저 내에서 주로 사용하며,
다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다.
또한 Node.js와 같은 런타임 환경과 같이 서버 사이드 네트워크 프로그래밍에도 사용되고 있다.
자바스크립트가 자바와 구문이 유사한 점도 있지만
이는 사실 두 언어 모두 C 언어의 기본 구문에 바탕을 뒀기 때문이고
자바와 자바스크립트는 직접적인 관련성이 없다.
